Python的常用第三方库整理

2019-03-09
Python

Python机器学习方向的第三方库有

Scikit-learn (专门用于机器学习的模块)

Theano (专门用于定义、优化、求值数学表达式适用于多维数组,特别适合做机器学习)

TensorFlow (世界上最受欢迎的开源机器学习框架,)

Caffe2

MxNet

Keras

PyTorch

Neon

Pandle

Python网络爬虫方向的第三方库

requests

scrapy

Python数据分析方向的第三方库

scipy

pandas

numpy

Python文本处理方向的第三方库

pdfminer

openpyxl

python-docx

python-pptx

beautifulsoup4

Python数据可视化方向的第三方库

matplotlib

mayavi

TVTK

Python Web开发方向的第三方库

Django

Flask

Pyramid

Python中文分词的第三方库

jieba

Pyserial (是硬件开发的第三方库)

requests (是处理HTTP请求的第三方库)

SymPy (是一个支持符号计算的Python第三方库)

TensorFlow (是Python语言的一套优秀的GUI图形库)

NLTK (是支持多种语言的自然语言处理Python第三方库)

WeRoBot (是一个微信公众号开发框架，也称为微信机器人框架)

matplotlib (主要进行二维图表数据展示，广泛用于科学计算的数据可视化)

MyQR (是一个能够产生基本二维码、艺术二维码和动态效果二维码的Python第三方库)
