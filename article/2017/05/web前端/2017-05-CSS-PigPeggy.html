用纯Css画小猪佩奇

2017-05-11
web前端
  • css

突发奇想，想用CSS来画个东西，画啥好勒？？又简单点的。。那就小猪佩奇吧，最近也比较火

代码大部分是使用border-radius方法绘制，如果用canvas的话会简单一点，代码就少挺多

/*先画个猪头*/
 .pigqi{
     position: absolute;
     margin-top:100px;
     margin-left: 40%;
     padding: 0;
 }

     position: absolute;
      z-index: 180;

     height: 300px;

     background:rgb(255,175,213);
 }
